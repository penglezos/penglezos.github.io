Englezos Kernel

I supported the following devices: Redmi Note 4 (codename: mido) and K20/MI 9T Pro (codename: raphael).

All of my kernels focused on stability/security by merging in the stable updates from kernel.org, any relevant updates from Qualcomm, and building with newer compilers to fix warnings during compilation.


Lite CSS

A fully responsive CSS framework that is small to provide good performance and saving time by providing a good base for any website. To keep it as minimalistic as possible this framework ignores the use of JavaScript to reduce loading and rendering times through possible delays caused by scripts.


Live Search

Live Search is a PHP search form that displays the result as you type.


Android Template

A basic template for Android custom ROMs created for PixelExperience redesign contest.


Scripts
